使用Hugo快速搭建个人博客网站
@ Huangkongsheng · Thursday, Sep 10, 2020 · 2 分钟阅读 · 更新于 Sep 10, 2020

比较多人习惯用Hexo、Hugo、Jekyll这些博客框架搭建自己的博客。本文详细介绍利用最新的Hugo来(基于Go语言的博客框架)在windows上搭建个人静态博客网站。

使用Hugo搭建博客前提：

电脑安装并配置好Git 在注册GitHub账号 掌握一定cmd和linux命令操作 掌握配置各种环境变量

第一步:安装Hugo

Hugo官方下载地址 下载下面的压缩包

下载完成后就解压放在你喜欢的地方，不用安装，就直接在windows配置环境变量就好了。 在cmd输入hugo version 出现如下信息就表示安装好了。

第二步：创建一个人博客文件夹

在当前目录右键打开git

输入：hugo new site myblog （myblog是自己的博客名） 打开myblog

第三步：下载并设置博客主题

进入Hugo博客主题（https://themes.gohugo.io/）下载库挑选自己喜欢的博客模板这里使用Prav模板为例点击Download

也可以点击Dome浏览一下效果 复制链接 切换到themes文件夹，然后输入： git clone https://github.com/pravin/hugo-theme-prav.git 克隆，这样直接克隆速度是比较慢点，可以借用gitee的仓库加快速度。 下载完后

第四步:本地启动博客

进入exampleSite文件夹，然后把里面的config.toml文件复制自己博客站点的根目录下，接点击替换就好。 输入一下命令：

hugo server -t m10c --buildDrafts

复制http://localhost:1313到自己的浏览器中，运行成功：

第五步：写一篇文章进行测试

命令行中键入hugo new post/firstblog.md 使用Typora打开写入内容（也可以使用其他软件打开比如：VC或者sublime） 重新运行

hugo server -t hugo-theme-prav --buildDrafts

想要配置自己的博客主题，可以看github上这个主题Hugo Theme Prav文档介绍（https://github.com/pravin/hugo-theme-prav），里面有说如何在config.toml文件中设置这款主题的样式

第六步:将个人博客部署到Github上

点击如下创建仓库 名称这里一定要跟你的用户名一样然后加.github.io,我这里报错是因为我已经过一个了。 在myblog目录下，cmd命令行中键入命令hugo --theme=hugo-theme-dream --baseUrl="https://huangkongsheng.github.io/" --buildDrafts

代码执行后就会在myblog文件夹下生成一个public文件夹

切换到public文件夹下 1、git init 2、git add . 3、git commit -m "我的博客第一次提交" 4、git remote add origin https://github.com/huangkongsheng/huangkongsheng.github.io.git 5、git push -u -origin master
